# <a name="ios-device-restrictions-settings-list-in-microsoft-intune"></a>Microsoft Intune 中的 iOS 设备限制设置

本文列举并介绍了可以为 iOS 设备配置的所有设备限制设置。 将这些设置添加到设备配置文件，然后使用 Microsoft Intune 将其分配或部署到 iOS 设备中。

## <a name="general"></a>常规

- **共享使用情况数据**：选择“阻止”可阻止设备将诊断和使用情况数据发送给 Apple。 “未配置”则允许发送此数据。
  - **诊断数据提交**：选择“阻止”可阻止用户更改“诊断和使用情况”中的诊断提交和应用分析设置（设备设置）。 若要使用此设置，设备必须处于监督模式 (iOS 9.3.2+)。 “未配置”则允许用户更改这些设备设置。
- **屏幕捕获**：选择“阻止”可阻止在设备上进行屏幕截图或屏幕捕获。 “未配置”则允许用户将屏幕内容作为图像捕获。
  - **通过 Classroom 应用远程观察屏幕(仅限受监督的设备)**：选择“阻止”可阻止 Classroom 应用远程观察设备上的屏幕。 若要使用此设置，设备必须处于监督模式 (iOS 9.3+)。 “未配置”则允许 Apple Classroom 应用查看屏幕。
  - **通过 Classroom 应用以静默方式观察屏幕(仅限受监督的设备)**：如果设置为“允许”，教师可使用 Classroom 应用以静默方式观察学生 iOS 设备的屏幕，学生不会察觉。 使用 Classroom 应用在类中注册的学生设备自动向该课程的教师授予权限。 “未配置”则阻止此功能。
- **不受信任的 TLS 证书**：选择“阻止”可阻止设备使用不受信任的传输层安全性 (TLS) 证书。 “未配置”则允许 TLS 证书。
- **企业应用信任**：选择“阻止”可删除设备上“设置”>“常规”>“配置文件和设备管理”中的“信任企业开发人员”按钮。 “未配置”则允许用户选择信任不是从应用商店下载的应用。
- **帐户修改(仅限受监督的设备)**：当设置为“阻止”时，用户无法从 iOS 设置应用更新特定于设备的设置。 例如，用户无法创建新的设备帐户，或更改用户名或密码。 “未配置”则允许用户更改这些设置。
  此功能同样适用于可从 iOS 设置应用（如邮件、联系人、日历、Twitter 等）进行访问的设置。 此功能不适用于具备不可从 iOS 设置应用进行配置的帐户设置的应用，例如，Microsoft Outlook 应用。
- **在设备设置中启用限制(仅限受监督的设备)**：选择“阻止”可阻止用户在设备设置中启用限制。 “未配置”则允许用户在设备上配置设备限制（如家长控制）。
- **使用设备上的擦除所有内容和设置选项(仅限受监督的设备)**：选择“阻止”，则用户无法使用设备上的擦除所有内容和设置选项。 “未配置”则授权用户对这些设置的访问权限。
- **修改设备名称(仅限受监督的设备)**：选择“阻止”，则无法更改设备名称。 “未配置”则允许用户更改设备的名称。
- **修改通知设置(仅限受监督的设备)**：选择“阻止”，则无法更改通知设置。 “未配置”则允许用户更改设备通知设置。
- **修改墙纸(仅限受监督的设备)**：“阻止”可阻止更改墙纸。 “未配置”则允许用户更改设备上的墙纸。
- **修改企业应用信任设置(仅限受监督的设备)**：“阻止”可阻止用户更改受监督设备上的企业应用信任设置。 “未配置”则允许用户信任不是从应用商店下载的应用。
- **配置文件更改(仅限受监督的设备)**：“阻止”可阻止设备上的配置文件更改。 “未配置”则允许用户安装配置文件。
- **激活锁(仅限受监督的设备)**：选择“允许”可在受监督的 iOS 设备上启用激活锁。 激活锁让重新激活丢失或被盗设备变得很难。
- **阻止应用删除(仅限受监督的设备)**：选择“阻止”可阻止用户删除应用。 “未配置”则允许用户从设备中删除应用。
- **阻止 USB 受限模式(仅限受监督的设备)**：选择“阻止”可禁用受监督设备上的 USB 受限模式。 USB 受限模式可阻止 USB 附件与已锁定一个多小时的设备交换数据。 “未配置”则允许 USB 受限模式。
- **强制自动日期和时间(仅限受监督的设备)**：“必需”可强制受监督设备自动设置日期和时间。 当设备连接移动电话网络或启用了位置服务的 Wi-fi 时，将更新设备的时区。
- **要求学生请求准许离开 Classroom 课程(仅限受监督的设备)**：“必需”可强制使用 Classroom 应用请求教师准许离开该课程。 仅适用于 iOS 11.3+。 “未配置”则不强制学生请求许可。
- **允许无线 PKI 更新**：“允许”可允许你的用户接收软件更新，而不用将其设备连接到计算机。
- **限制广告跟踪**：选择“限制”可禁用设备广告标识符。 “未配置”则使其保持启用状态。
- **阻止 VPN 创建(仅限受监督的设备)**：“阻止”可阻止用户创建 VPN 配置设置。 “未配置”则允许用户在设备上创建 VPN。

## <a name="configurations-requiring-supervision"></a>配置需要监督

仅在通过 Apple 设备注册计划或使用 Apple Configurator 初次设置设备时，才可启用 iOS 受监督模式。 启用受监督模式后，Intune 可使用以下功能配置设备：

- 应用锁定（单应用模式） 
- 全局 HTTP 代理 
- 激活锁绕过 
- 自治单应用模式 
- Web 内容筛选器 
- 设置背景和锁屏界面 
- 无提示应用推送 
- 始终可用 VPN 
- 允许以独占方式安装托管应用 
- iBookstore 
- iMessage 
- 游戏中心 
- AirDrop 
- AirPlay 
- 主机配对 
- 云同步 
- Spotlight 搜索 
- Handoff 
- 擦除设备 
- 限制 UI 
- 通过 UI 安装配置文件 
- 新闻 
- 键盘快捷方式 
- 密码修改 
- 设备名更改 
- 自动应用下载 
- 企业应用信任更改 
- Apple Music 
- Mail Drop 
- 与 Apple Watch 配对 

> [!NOTE]
> Apple 已确认某些设置将于 2019 年迁移到“仅受监督”模式。 我们建议在使用这些设置时即考虑此更改，而不是等待 Apple 将它们迁移到“仅受监督”模式：
> - 由最终用户安装的应用
> - 应用删除
> - FaceTime
> - Safari
> - iTunes
> - 成人内容
> - iCloud 文档和数据
> - 多玩家游戏
> - 添加游戏中心好友
> - Siri

## <a name="password"></a>密码

- **密码**：需要最终用户输入密码才能访问设备。 “未配置”则允许用户访问设备，而无需输入密码。
  - **简单密码**：选择“阻止”，进而需要更复杂的密码。 “未配置”则允许使用简单密码，如 `0000` 和 `1234`。
  - **所需的密码类型**：选择你的组织需要的密码类型。 选项包括：
    - **设备默认值**
    - **数字**
    - **字母数字**
  - **密码中的非字母数字字符数**：输入密码中必须包含的符号字符（如 `#` 或 `@`）数。
  - **最短密码长度**：输入用户必须输入的最短长度（介于 4 到 14 个字符之间）。
  - **擦除设备前登录失败的次数**：输入在擦除设备前允许的登录失败次数（介于 1 到 11 之间）。
  - **屏幕锁定后要求提供密码前的最大分钟数**<sup>1</sup>：输入用户必须重新输入密码前设备保持空闲状态的时间。 如果输入的时间大于设备上当前设置的时间，则设备将忽略输入的时间。 在 iOS 8.0 及更高版本的设备上受支持。
  - **屏幕锁定前的最大不活动分钟数**<sup>1</sup>：输入在屏幕锁定前在设备上允许的最大不活动分钟数。 如果输入的时间大于设备上当前设置的时间，则设备将忽略输入的时间。
  - **密码过期(天数)**：输入必须更改设备密码前的天数。
  - **防止重用以前的密码**：输入必须使用多少个新密码后才能重用旧密码。
  - **指纹解锁**：选择“阻止”可阻止使用指纹解锁设备。 “未配置”则允许用户使用指纹解锁设备。
- **密码修改(仅限受监督的设备)**：选择“阻止”可阻止更改、添加或删除密码。 阻止此功能后，受监督设备上将忽略对密码限制的更改。 “未配置”则允许添加、更改或删除密码。
  - **指纹修改(仅限受监督的设备)**：“阻止”可阻止用户更改、添加或删除 TouchID 指纹。 “未配置”则允许用户在设备上更新 TouchID 指纹。
- **阻止密码自动填充(仅限受监督的设备)**：选择“阻止”可阻止在 iOS 上使用自动填充密码功能。 选择“阻止”还可执行以下操作：
  - 系统不会提示用户在 Safari 或任何应用中使用已保存的密码。
  - 自动强密码处于禁用状态，不建议用户使用强密码。

  “未配置”则允许这些功能。

- **阻止密码接近度请求(仅限受监督的设备)**：选择“阻止”，则用户的设备不会从附近的设备请求密码。 “未配置”则允许这些密码请求。
- **阻止密码共享(仅限受监督的设备)**：“阻止”可阻止使用 AirDrop 在设备之间共享密码。 “未配置”则允许共享密码。

<sup>1</sup>配置“屏幕锁定前的非活动状态最大分钟数”和“屏幕锁定后要求提供密码前的最大分钟数”时，它们会依次应用。 例如，如果设置的两个设置的值均为“5”分钟，屏幕在五分钟后将自动关闭，然后再过五分钟后该设备将锁定。 但是，如果用户手动关闭屏幕，第二个设置将立即应用。 在相同的示例中，用户关闭屏幕后，该设备将在五分钟后锁定。

## <a name="locked-screen-experience"></a>锁定屏幕体验

- **在设备锁定时访问控制中心**：选择“阻止”可阻止在设备锁定时访问控制中心应用。 “未配置”则允许用户在设备锁定时访问控制中心应用。
- **设备锁定时的通知**：“阻止”可在设备锁定时阻止对通知的访问。 “未配置”则允许用户在不解锁设备的情况下访问通知。
- **设备锁定时的钱包通知**：“阻止”可在设备锁定时阻止对钱包应用的访问。 “未配置”则允许用户在设备锁定时访问钱包应用。
- **在设备锁定时查看“今日”视图**：“阻止”可阻止用户在设备锁定时查看“今日”视图。 “未配置”则允许用户在设备锁定时查看“今日”视图。

## <a name="app-store-doc-viewing-gaming"></a>App Store、文档查看和游戏

- **App Store**：“阻止”可阻止访问受监督设备上的 App Store。 “未配置”则允许访问。
  - **从 App Store 安装应用(仅限受监督的设备)**：选择“阻止”可阻止在设备主屏幕使用 App Store。 最终用户可以继续使用 iTunes 或 Apple Configurator 安装应用。 “未配置”则允许在主屏幕使用 App Store。
  - **自动下载应用(仅限受监督的设备)**：选择“阻止”可阻止自动下载其他设备上购买的应用。 它不会影响对现有应用的更新。 “未配置”则允许在设备上下载其他 iOS 设备上购买的应用。
- **输入密码以访问 App Store**：“必需”，用户输入密码后才能访问 App Store。 “未配置”则允许无需输入密码即可访问 App Store。
- **应用内购买**：选择“阻止”可阻止从应用商店进行应用内购买。 “未配置”则允许在运行的应用中产生应用商店购买行为。
- **限制级 iTunes 音乐、播客或新闻内容(仅限受监督的设备)**：选择“阻止”可阻止限制级 iTunes 音乐、播客或新闻内容。 “未配置”则允许设备访问存储中认定为成人的内容。
- **从 iBook 商店下载标记为“成人作品”的内容**：选择“阻止”可阻止用户从 iBook 商店下载标记为“成人作品”的媒体。 “未配置”则允许用户下载类别为“成人作品”的书籍。
- **在非托管应用中查看企业文档**：“阻止”可阻止在非托管应用中查看非企业文档。 “未配置”则允许在任何应用中查看企业文档。 例如，你想要阻止用户将文件从 OneDrive 应用保存到 Dropbox。 将此设置配置为“阻止”。 设备收到策略后（例如在重新启动后），将不再允许保存。
- **在企业应用中查看非企业文档**：“阻止”可阻止在企业应用中查看非企业文档。 “未配置”则允许在公司托管的应用中查看任何文档。
- **将 AirDrop 视为非托管目标**：“必需”可强制将 AirDrop 视为非托管拖放目标。 它将阻止托管应用使用 Airdrop 发送数据。 
- **添加 Game Center 好友(仅限受监督的设备)**：“阻止”可阻止用户添加 Game Center 好友。 “未配置”则允许用户在 Game Center 中添加好友。
- **Game Center (仅限受监督的设备)**：“阻止”可阻止使用 Game Center 应用。 “未配置”则允许在设备上使用 Game Center 应用。
- **多玩家游戏(仅限受监督的设备)**：选择“阻止”可阻止多玩家游戏。 “未配置”则允许用户在设备上玩多玩家游戏。
- **分级区域**：选择要用于允许的下载的分级区域。 然后，选择“电影”和“电视节目”的允许分级。
- **应用**：选择用户可下载的应用的允许年龄分级，或者可以选择“允许所有应用”。

  - **Siri 从 Internet 查询用户生成的内容(仅限受监督的设备)**：“阻止”可阻止 Siri 访问网站以回答问题。 “未配置”则允许 Siri 从 Internet 访问用户生成的内容。
- **Apple News (仅限受监督的设备)**：选择“阻止”可阻止访问设备上的 Apple News 应用。 “未配置”则允许使用 Apple News 应用。
- **iBooks Store (仅限受监督的设备)**：“阻止”可阻止访问 iBooks Store。 “未配置”则允许用户从 iBooks Store 浏览和购买书籍。
- **设备上的邮件应用(仅限受监督的设备)**：选择“阻止”，则用户无法使用设备上的邮件应用。 “未配置”则允许使用邮件应用发送和接收短信。
- **播客(仅限受监督的设备)**：“阻止”可阻止用户使用播客应用。 “未配置”则允许使用播客应用。
- **音乐服务(仅限受监督的设备)**：“阻止”可将音乐应用还原为经典模式并禁用音乐服务。 “未配置”则允许使用 Apple 音乐应用。
- **iTunes Radio 服务(仅限受监督的设备)**：“阻止”可阻止用户使用 iTunes Radio 应用。 “未配置”则允许使用 iTunes Radio 应用。
- **更改“查找我的好友”应用设置(仅限受监督的设备)**：“阻止”可阻止更改“查找我的好友”应用设置。 “未配置”则允许用户更改“查找我的好友”应用的设置。
- **Spotlight 搜索从 Internet 返回结果(仅限受监督的设备)**：“阻止”可阻止 Spotlight 从 Internet 搜索返回任何结果。 “未配置”则允许 Spotlight 搜索连接到 Internet 以提供搜索结果。
- **阻止从设备中删除系统应用(仅限受监督的设备)**：选择“阻止”可禁用从设备中删除系统应用的功能。 “未配置”则允许用户删除系统应用。

## <a name="restricted-apps"></a>受限制的应用

在受限制的应用列表中，可以配置以下列表之一：

- **禁止的应用**：不想在设备上安装的不由 Intune 管理的应用列表。 如果用户安装此列表中的某个应用，Intune 会通知你。
- **批准的应用**：允许用户安装的应用列表。 为了保持兼容性，用户不得安装其他应用。 自动允许由 Intune 托管的应用。 如果用户安装此列表中的某个应用，Intune 会通知你。

若要将应用添加到这些列表，可以：

- 添加所需应用的 iTunes App Store URL。 例如，若要添加 Microsoft 工作文件夹应用，请输入 `https://itunes.apple.com/us/app/work-folders/id950878067?mt=8`。

  若要查找应用的 URL，请打开 iTunes App Store，并搜索该应用。 例如，搜索 `Microsoft Remote Desktop` 或 `Microsoft Word`。 选择应用并复制 URL。

  还可使用 iTunes 查找应用，然后使用“复制链接”任务获取应用 URL。

- 导入包含应用详细信息的 CSV 文件，包括 URL。 使用 <app url>、<app name>、<app publisher> 格式。 或，导出包含相同格式的受限应用列表的现有列表。

> [!IMPORTANT]
> 必须将使用受限制的应用设置的设备配置文件分配到用户组。

## <a name="show-or-hide-apps-supervised-only"></a>显示或隐藏应用（仅限被监督的设备）

在“显示或隐藏应用”列表中，可以在运行 iOS 9.3 或更高版本的受监督设备上配置以下列表之一。

- **隐藏应用**：输入对用户隐藏的应用列表。 用户无法查看，或打开这些应用。
- **可见应用**：输入用户可以查看和启动的应用列表。 无法查看或启动其他应用。

若要将应用添加到这些列表，可以：

- 添加所需应用的 iTunes App Store URL。 例如，若要添加 Microsoft 工作文件夹应用，请输入 `https://itunes.apple.com/us/app/work-folders/id950878067?mt=8`。

  若要查找应用的 URL，请打开 iTunes App Store，并搜索该应用。 例如，搜索 `Microsoft Remote Desktop` 或 `Microsoft Word`。 选择应用并复制 URL。

  还可使用 iTunes 查找应用，然后使用“复制链接”任务获取应用 URL。

- 导入包含应用详细信息的 CSV 文件，包括 URL。 使用 <app url>、<app name>、<app publisher> 格式。 或，导出包含相同格式的受限应用列表的现有列表。

## <a name="wireless"></a>无线

- **数据漫游**：选择“阻止”可阻止通过移动电话网络进行数据漫游。 “未配置”则允许设备在处于移动电话网络时进行数据漫游。
- **漫游时进行全局后台获取**：选择“阻止”可阻止在通过移动电话网络漫游时使用全局后台获取功能。 “未配置”则允许设备在通过移动电话网络漫游时提取数据，例如电子邮件。
- **语音拨号**：选择“阻止”可阻止用户在设备上使用语音拨号功能。 “未配置”则允许在设备上使用语音拨号。
- **语音漫游**：选择“阻止”可阻止通过移动电话网络进行语音漫游。 “未配置”则允许设备在处于移动电话网络时进行语音漫游。
- **更改应用的移动电话网络数据使用情况设置(仅限受监督的设备)**：选择“阻止”可阻止更改应用的移动电话网络数据使用情况设置。 “未配置”则允许用户控制允许哪些应用使用移动电话网络数据。
- **个人热点**：“阻止”可阻止将设备用作个人热点。 此设置可能与某些运营商不兼容。 “未配置”则允许此功能。
- **加入仅使用配置文件的 Wi-Fi 网络(仅限受监督的设备)**：“必需”可强制设备仅使用通过 Intune 配置文件设置的 Wi-Fi 网络。 “未配置”则允许设备使用其他 Wi-Fi 网络。
- **移动网络的使用规则(仅限托管应用)**：定义托管应用可在移动网络中使用的数据类型。 选项包括：
  - **阻止使用移动电话数据**：对“所有托管应用”或“选择特定应用”阻止使用移动电话数据。
  - **阻止漫游时使用移动电话数据**：对“所有托管应用”或“选择特定应用”阻止漫游时使用移动电话数据。

- **AirDrop (仅限受监督的设备)**：“阻止”可阻止在设备上使用 AirDrop。 “未配置”则允许使用 AirDrop 功能与附近的设备交换内容。

- **修改蓝牙(仅限受监督的设备)**：“阻止”可阻止最终用户更改设备上的蓝牙设置。 “未配置”则允许用户更改这些设置。
- **通过主机配对来控制 iOS 设备可与之配对的设备(仅限受监督的设备)**：“未配置”可允许主机配对，以使管理员能够控制 iOS 设备可与之配对的设备。 “阻止”则阻止主机配对。
- **需要 AirPlay 传出请求配对密码**：“必需”，当用户使用 AirPlay 将内容流式传输到其他 Apple 设备时，需要提供配对密码。 “未配置”则允许用户在不输入密码的情况下使用 AirPlay 流式传输内容。
- **阻止 AirPrint (仅限受监督的设备)**：选择“阻止”可阻止在设备上使用 AirPrint 功能。 “未配置”则允许用户使用 AirPrint。
  - **阻止密钥链中 AirPrint 凭据的存储(仅限受监督的设备)**：“阻止”可阻止使用设备用户名和密码的密钥链存储。 “未配置”则允许将 AirPrint 用户名和密码存储在密钥链应用中。
  - **针对 AirPrint 要求受信任 TLS 证书(仅限受监督的设备)**：“必需”可强制设备将受信任的证书用于 TLS 打印通信。
  - **阻止 iBeacon 发现 AirPrint 打印机(仅限受监督的设备)**：选择“阻止”可防止恶意 AirPrint 蓝牙信标对网络流量进行钓鱼。 “未配置”则允许在设备上公布 AirPrint 打印机。

- **预测键盘(仅限受监督的设备)**：“未配置”可允许使用预测键盘建议用户可能想要使用的字词。 “阻止”则阻止此功能。
- **自动更正(仅限受监督的设备)**：“未配置”可允许设备自动更正拼写错误的词。 “阻止”则阻止使用自动更正。
- **键盘拼写检查(仅限受监督的设备)**：“未配置”可允许在设备上使用拼写检查器。 “阻止”则允许使用拼写检查器。
- **键盘快捷方式(仅限受监督的设备)**：“未配置”可允许在设备上使用键盘快捷方式。 “阻止”则阻止用户使用键盘快捷方式。
- **听写(仅限受监督的设备)**：“阻止”可阻止用户通过语音输入输入文本。 “未配置”则允许用户使用听写输入。

## <a name="cloud-and-storage"></a>云和存储

- **备份到 iCloud**：“未配置”可允许用户将设备备份到 iCloud。 “阻止”则阻止用户将设备备份到 iCloud。
- **将文档同步到 iCloud (仅限受监督的设备)**：“未配置”可允许将文档和键值同步到 iCloud 存储空间。 “阻止”则阻止 iCloud 同步文档和数据。
- **将照片流同步到 iCloud**：“未配置”可允许用户在其设备上启用“我的照片流”以将照片同步到 iCloud 并在所有用户设备上可用。 “阻止”则阻止将照片流同步到 iCloud。
- **加密的备份**：“必需”，设备备份必须加密。
- **iCloud 照片库**：设置为“阻止”可禁止使用 iCloud 照片库将照片和视频存储在云中。 会从设备中删除尚未从 iCloud 照片库完全下载到设备的所有照片。 “未配置”则允许使用 iCloud 照片库。
- **通过托管的应用同步到云**：“未配置”可允许通过 Intune 管理的应用将数据同步到用户的 iCloud 帐户。 “阻止”则阻止将此数据同步到 iCloud。
- **共享照片流**：选择“阻止”可在设备上禁用“iCloud 照片共享”。 “未配置”则允许共享照片流。
- **活动延续**：“未配置”可允许用户在其他 iOS 或 macOS 设备上继续进行在某台 iOS 设备上开始的工作（切换）。 “阻止”则阻止此切换。
- **阻止 iCloud 密钥链同步**：选择“阻止”可禁止将密钥链中存储的凭据同步到 iCloud。 “未配置”则允许用户同步这些凭据。
- **阻止企业簿备份**：选择“阻止”可阻止用户备份企业簿。 “未配置”则允许用户备份这些企业簿。
- **阻止企业簿元数据同步(备注和重要内容)**：“阻止”可阻止同步企业簿中的备注和重要内容。 “未配置”则允许同步。

## <a name="autonomous-single-app-mode-supervised-only"></a>自治单应用模式（仅限监督的应用）

使用这些设置配置 iOS 设备，以自主单一应用模式运行特定应用。 配置此模式并运行应用时，将锁定设备。 因此该设备只能运行该应用。 例如，添加允许用户在设备上进行测试的应用。 应用的操作完成或删除此策略时，设备将恢复正常状态。

若要添加应用，可以：

- 输入“应用名称”和“应用捆绑 ID”，然后选择“添加”。 [适用于内置 iOS 应用的捆绑 ID 引用](#bundle-id-reference-for-built-in-ios-apps)（在本文中）包括具有其 ID 的某些应用。
- 导入包含应用名称及其捆绑 ID 的列表的 CSV 文件。 或，导出包含应用的现有列表。

## <a name="kiosk-supervised-only"></a>展台（仅限被监督的设备）

- **要在展台模式下运行的应用**：选择要在展台模式下运行的应用的类型。 选项包括： 
  - **应用商店应用**：输入 iTunes App Store 中的应用的 URL
  - **托管应用**：选择已添加到 Intune 的应用
  - **内置应用**：输入内置应用的[捆绑 ID](#bundle-id-reference-for-built-in-ios-apps)

- **辅助触控**：“必需”，需要在设备上启用“辅助触控”辅助功能设置。 此功能可帮助用户执行可能难以执行的屏幕手势。 “未配置”则不会在展台模式下运行或启用此功能。
- **反色**：“必需”，需要启用“反色”辅助功能设置，以便视力残障人士可以更改显示屏幕。 “未配置”则不会在展台模式下运行或启用此功能。
- **单声道音频**：“必需”，需要在设备上启用“单声道音频”辅助功能设置。 “未配置”则不会在展台模式下运行或启用此功能。
- **VoiceOver**：“必需”，需要在设备上启用“VoiceOver”辅助功能设置，以便朗读屏幕上的文本。 “未配置”则不会在展台模式下运行或启用此功能。
- **缩放**：“必需”，需要在设备上启用缩放设置，以便用户能够通过触摸在屏幕上进行放大。 “未配置”则不会在展台模式下运行或启用此功能。
- **自动锁定**：“允许”可允许自动锁定设备。 “未配置”则会禁用此功能。
- **响铃开关**：“允许”可启用设备上的响铃（静音）开关。 “未配置”则会禁用此功能。
- **屏幕旋转**：“允许”可允许在用户旋转设备时更改屏幕方向。 “未配置”则会禁用此功能。
- **屏幕睡眠按钮**：选择“允许”可禁用设备上的屏幕睡眠唤醒按钮。 “未配置”则会启用此功能。
- **触控**：“阻止”可禁用设备上的触摸屏。 “未配置”则允许用户使用触摸屏。
- **音量按钮**：“允许”可允许使用设备上的音量按钮。 “未配置”则会禁用音量按钮。
- **辅助触控**：“允许”可允许用户使用辅助触控功能。 “未配置”则会禁用此功能。
- **反色控制**：“允许”可启用反色调整，以允许用户调整反色功能。 “未配置”则会禁用此功能。
- **朗读所选文本**：“允许”可在设备上启用“朗读选择”辅助功能设置为。 此功能可朗读用户选择的文本。 “未配置”则会禁用此功能。
- **VoiceOver 控制**：“允许”可允许更改画外音，以允许用户更新 VoiceOver 功能，例如朗读屏幕上文本的速度。 “未配置”则阻止 voiceover 更改。
- **缩放控件**：“允许”可允许用户所做的缩放更改。 “未配置”则阻止缩放更改。

> [!NOTE]
> 必须使用 Apple Configurator 工具或 Apple 设备注册程序将设备置于监督模式后才能为 iOS 设备配置展台模式。 有关如何使用 Apple 配置器工具，请参阅 Apple 的指南。
> 如果在分配配置文件之后安装输入的 iOS 应用，则设备将在重新启动后才会进入展台模式。

## <a name="safari"></a>Safari

- **Safari (仅限受监督的设备)**：“阻止”可阻止在设备上使用 Safari 浏览器。 “未配置”则允许用户使用 Safari 浏览器。
- **自动填充**：“阻止”可禁用设备上的 Safari 中的自动填充功能。 “未配置”则允许用户更改 Web 浏览器中的自动完成设置。

## <a name="domains"></a>Domains

### <a name="unmarked-email-domains"></a>未标记的电子邮件域

在“电子邮件域 URL”中，向列表添加一个或多个 URL。 当最终用户从所输入的域以外的域接收电子邮件时，该电子邮件在 iOS 邮件应用中被标记为不受信任。

### <a name="managed-web-domains"></a>托管的 Web 域

在“Web 域 URL”中，向列表添加一个或多个 URL。 从所输入的域下载文档时，这些文档会被视为托管。 此设置仅适用于使用 Safari 浏览器下载的文档。

### <a name="safari-password-autofill-domains"></a>Safari 密码自动填充域

在“域 URL”中，向列表添加一个或多个 URL。 用户仅能保存来自此列表中的 URL 的 Web 密码。 此设置仅适用于 Safari 浏览器以及监督模式下的 iOS 9.3 和更高版本的设备。 如未指定任何 URL，则可从所有网站保存密码。
