# <a name="microsoft-intune-protected-apps"></a>受 Microsoft Intune 保护的应用  

本主题列出的应用是常用于 Microsoft Intune 的受支持合作伙伴应用和 Microsoft 应用。 受 Intune 保护的应用启用了一组丰富的移动应用保护策略。 这些应用可便于：

- 限制“复制和粘贴”和“另存为”功能
- 将 Web 链接配置为在安全的 Microsoft 浏览器内打开
- 启用多标识使用和应用级条件访问
- 应用数据丢失防护策略，无需管理用户设备
- 启用应用保护，无需注册
- 在使用第三方 EMM 工具托管的设备上启用应用保护

> [!NOTE]
> 对于客户端业务线应用，可使用 [Intune 应用软件开发工具包](../developer/app-sdk.md) (SDK) 或 [iOS 应用包装工具](../developer/app-wrapper-prepare-ios.md)和 [Android 应用包装工具](../developer/app-wrapper-prepare-android.md)纳入移动应用管理。
