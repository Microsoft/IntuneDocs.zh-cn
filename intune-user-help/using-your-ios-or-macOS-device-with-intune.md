# <a name="using-your-ios-or-macos-device-with-intune"></a>通过 Intune 使用 iOS 或 macOS 设备

## <a name="supported-versions"></a>支持的版本

Intune 当前支持 iOS 8.0 及更高版本。 它也支持 macOS 10.9 Mavericks 及更高版本。
