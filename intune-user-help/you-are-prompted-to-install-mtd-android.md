# <a name="you-need-to-install-mobile-threat-defense-on-your-android-device"></a>需要在 Android 设备上安装移动威胁防御

在可访问工作之前，IT 管理员要求你安装移动威胁防御应用，该应用会查找潜在安全威胁来帮助保护你的设备。 设备上显示的提示可能有所不同，具体取决于 IT 管理员对移动威胁防御的设置。

* [Lookout for Work](you-are-prompted-to-install-lookout-for-work-android.md)
* [Skycure](you-are-prompted-to-install-skycure-android.md)

**如果安装不成功**

有时安装可能会因不可控的技术问题而失败。 如果发生此情况，请与你的 IT 管理员联系。 有关联系信息，请查看[公司门户网站](http://portal.manage.microsoft.com)。
