# <a name="your-device-is-noncompliant"></a>设备不合规

设备不符合至少一条组织的安全策略，但无法立即显示错误。  

在公司门户应用中，点击“检查合规性”。 公司门户应用将显示导致设备不合规的原因。

仍需要帮助？ 请与 IT 管理员联系。 有关联系信息，请查看[公司门户网站](http://portal.manage.microsoft.com)。
