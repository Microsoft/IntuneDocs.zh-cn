
  - Apple iOS 7.1 及更高版本
  - Google Android 4.0 及更高版本（包括 Samsung KNOX SDK 4.0 及更高版本）
  - Windows Phone 8.0 及更高版本
  - Windows RT 和 Windows 8.1 RT
  - 运行 Windows 8.1 的电脑
  - 运行 Windows 10（家庭版、专业版、教育版和企业版）的电脑
  - Mac OS X 10.9 及更高版本
