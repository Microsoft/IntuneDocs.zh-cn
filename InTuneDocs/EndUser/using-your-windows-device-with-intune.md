# 通过 Intune 使用 Windows 设备

本文包含用户可在其 Intune 托管的设备上执行的所有任务的链接。 IT 管理员可以参考此信息以查看用户执行各种任务时必须采用的步骤，这些任务按以下类别分组：
- 公司门户应用安装和 Intune 注册
- 注册后管理任务
- 合规性问题
