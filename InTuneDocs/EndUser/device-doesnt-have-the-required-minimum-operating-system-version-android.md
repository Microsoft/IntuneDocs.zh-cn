# 设备没有必需的最低操作系统版本

Android 设备不满足 IT 部门要求的最低操作系统版本。 更新设备之前，通过点击“设置” &gt; “关于手机”检查当前操作系统的版本，然后通过点击“设置”检查你的设备是否有可用更新，然后查找更新选项。
