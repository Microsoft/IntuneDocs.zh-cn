# 设备不符合最高操作系统版本

Android 设备不符合有关最高操作系统版本的 IT 管理员策略。 请联系你的 IT 管理员获取帮助。 有关他们的联系信息，请查看[公司门户网站](http://portal.manage.microsoft.com)。
