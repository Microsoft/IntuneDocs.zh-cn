# 安装你的公司虚拟专用网络 (VPN)

如果你的 IT 管理员配置了 VPN 应用程序来允许你连接到公司资源，则你的 Android 设备上将显示一个表明需要安装 VPN 应用的通知。 按照以下步骤安装 VPN 应用：

1.  下拉通知抽屉，然后点击“点击以安装此必需应用”

2.  在“Play Store”****中，单击“安装”****并按照提示安装应用。

3.  点击“安装公司 VPN 配置文件”****并按照提示接受并激活应用。
