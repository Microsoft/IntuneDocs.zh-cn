# 如何重新连接受威胁的 Android 设备
你的 IT 管理员检测到你正在使用取得 root 权限的设备。 此配置可能危害安全性，并可能会威胁到你的公司资源。

请联系你的 IT 管理人员帮助你重新连接设备。
