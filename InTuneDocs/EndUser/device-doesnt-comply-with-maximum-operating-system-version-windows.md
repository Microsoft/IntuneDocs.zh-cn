# Windows 设备不符合最高操作系统版本

IT 管理员设置了允许你在 Windows 设备上使用的最高操作系统版本，但你的设备不符合改设置。 若要了解适用于你设备的允许的最高操作系统版本，需要与 IT 管理员联系。 有关他们的联系信息，请查看[公司门户网站](http://portal.manage.microsoft.com)。
