# 错误评估设备
您的设备找不符合其中一个，或在计算此条件的设备的更多公司的策略和错误时发生。

要解决此问题，请等待几分钟，然后点击 **检查符合性**。

如果问题仍然存在，请与您的 IT 管理员联系。
