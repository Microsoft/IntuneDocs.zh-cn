# 设置或更改密码

如果 Windows 设备与 IT 管理员设置的密码策略不相符，则会出现一条消息，其中简要描述了该问题并指出使密码符合要求所需的操作。

IT 管理员可以设置多个密码要求。 例如，密码可能必须要达到一定数量的字符，并且可能必须是数字和字母的组合。 由于 IT 管理员可能会设置多个要求，因此这些说明只介绍如何设置或更改密码。 而不对每种可能组合的要求进行解释。

若在设置符合所有密码要求的密码时遇到问题，请联系 IT 管理员获得帮助。

## 在 Windows Phone 8.1 上更改密码

1. 在“主页”屏幕上，选择“设置”，然后选择“锁屏”。
2. 向下滚动到“密码”部分，然后选择“更改密码”。
3. 在“当前密码”框中输入当前密码。
4. 在“新密码”框中输入新密码，随后在“确认密码”框中再次输入该密码。
4. 选择“完成”。

## 在 Windows Phone 8.1 上设置密码

1. 在“主页”屏幕上，选择“设置”，然后选择“锁屏”。
2. 向下滚动到**密码**，然后滑动**密码**栏到**开启**。
3. 在“新密码”框中输入新密码，随后在“确认密码”框中再次输入该密码。
4. 选择“完成”。

仍需要帮助？ 请与 IT 管理员联系。 有关他们的联系信息，请查看[公司门户网站](http://portal.manage.microsoft.com)。
