# 设备符合性状态未知

如果你的设备显示“符合性状态未知”的消息，这表示 Intune 正尝试检查你的设备是否符合公司策略。 执行此检查时，可能会阻止你访问公司资源，如电子邮件。 请等待几分钟直到完成符合性检查，然后再次尝试访问公司资源。 

如果仍然无法访问公司资源，请与你的 IT 管理员联系。 有关他们的联系信息，请查看[公司门户网站](http://portal.manage.microsoft.com)。
